# Puzzle

Jigsaw puzzle game built with C++ Builder 6.0 as an engineering degree project. Features save/load system, high scores, MIDI audio, and animated intro.

## Features

- Multiple game screens: intro, game loading, gameplay, results, and high scores
- Save and load game progress
- MIDI audio playback via MCI interface
- Custom stack-based data structure for puzzle piece management
- Player profiles and leaderboard
